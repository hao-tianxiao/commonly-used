克隆现有存储库$ git clone ssh://user@domain.com/repo.git
创建一个新的本地存储库$ git init
将所有当前更改添加到下一个提交$ git add .
提交跟踪文件中的所有本地更改$ git commit -a
显示所有提交，从最新开始$ git log
下载更改并直接合并/集成到HEAD中$ git pull
在远程上发布本地更改$ git push
